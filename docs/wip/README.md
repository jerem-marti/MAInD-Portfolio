# Work-in-progress docs

Living, ongoing working documents that are tracked in the repo but are **not** polished reference
material. Unlike the published guides in `docs/`, these are review notes and journals that evolve as
the work does.

Current contents:

- **`FR-REVIEW.md`** — French translation review: glossary, voice rules, verbatim-keep list, and the
  per-file checklist. French is owner-reviewed; this is the reference for that review.
- **`FR-DECALQUAGE.md`** — journal of the French regeneration pass (regenerate idiomatic French from
  the English source rather than preserve the existing calque).

See `docs/i18n.md` for how the locale system works.
