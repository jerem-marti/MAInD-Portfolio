# Local scratch (not committed)

This folder is an owner-only local working area. **Its contents are gitignored** — only this
README is tracked, so the folder always exists on a fresh clone.

Drop anything here that should stay on your machine and never reach the public repo: throwaway
notes, backups, exported data, experiment files. Nothing in here (other than this README) is
committed.

For working docs you *do* want tracked (review notes, ongoing journals, design specs), use
`docs/wip/` (tracked WIP), `docs/superpowers/specs/` and `docs/superpowers/plans/` (tracked), or the
published guides in `docs/`.
